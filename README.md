# Shyam Paridhan - E-commerce Website

A modern, responsive e-commerce website for Shyam Paridhan clothing store.

## Features

- **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- **Modern UI** - Clean and professional design with smooth animations
- **Product Showcase** - Featured products with ratings, prices, and quick actions
- **Category Navigation** - Easy browsing by Men's, Women's, and Kids collections
- **Shopping Cart** - Add to cart functionality with live counter
- **Wishlist** - Save favorite products
- **Newsletter Subscription** - Email collection for marketing
- **Social Media Integration** - Links to all social platforms

## Quick Start

1. Open `index.html` in your web browser
2. That's it! No build process required.

## Customization

### Update Images
Replace placeholder images in `index.html`:
- Hero section: Line 42
- Category cards: Lines 56, 62, 68
- Product images: Lines 81, 107, 133, 159

### Update Colors
Edit CSS variables in `styles.css` (lines 9-15):
```css
--primary-color: #2c3e50;
--secondary-color: #e74c3c;
```

### Update Content
- Store name: Search for "SHYAM PARIDHAN" in `index.html`
- Contact info: Update footer section (lines 240-244)
- Products: Modify product cards starting at line 75
